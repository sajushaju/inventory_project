Inventory Project – Django DRF
-------------------------------

This is a Django REST Framework project for managing products in an inventory.
It supports full CRUD operations and comes with unit tests for each API endpoint.

Features
-------------

Product CRUD API (Create, Read, Update, Delete)

Unit tests for all endpoints

PostgreSQL as the database
create your DB with this name  nventory_db
Postman screenshots included

Project Setup
--------------
1. Clone the repository
        git clone  https://github.com/sajushaju/inventory_project.git
         cd inventory_project

2.Create and activate virtual environment
        python -m venv venv
        venv\Scripts\activate  ( # Windows)
3.Install dependencies
  pip install -r requirements.txt
4. create your DB with this name  nventory_db
5. Setup environment variables (.env file for holding database details)
6.Run migrations
7.Run the server
